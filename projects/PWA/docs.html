User guide

Features and installation

Pocket Planetarium is a Progressive Web App that teaches solar-system facts through short lessons, images, spoken audio, and a quiz. The app stores explored lessons in the browser so students can return later.

  • The lesson menu is populated from src/data/solar-system.json.
  • The manifest makes the app installable on supported browsers.
  • The service worker caches the app shell, JSON, and local image assets for offline use.
  • The audio button uses browser speech synthesis by default, and can also play an audio file if a lesson points to one.

How to install

Open the app from a web server, then use the Install button or the browser’s “Add to Home Screen” option. After the first visit, the cached app can reopen without a network connection.

Programmer guide

Changing the JSON dataset

Edit src/data/solar-system.json to change the learning set. Keep the top-level topics array and give each lesson these fields: id, title, description, image, audio, distance, orbit, type, caption, and quiz.

For audio, use "tts" to let the app read the lesson aloud, or replace it with a local audio path such as src/assets/audio/earth.mp3. Add any new image or audio files to the service worker cache list so they are available offline.
